Barba.BaseTransition

BaseTransitionを拡張します。


メンバ変数

Barba.BaseTransition.newContainer

type : HTMLElement

Barba.BaseTransition.newContainerLoading

type : Promise

Barba.BaseTransition.oldContainer

type : HTMLElement

関数

Barba.BaseTransition.done()

この関数は遷移終了後すぐに呼び出す必要があります。


Barba.BaseTransition.extend(newObject)

オブジェクトを拡張するヘルパーです。 戻り値はobjectです。

newObject : object

Barba.BaseTransition.start()

Transitionのコンストラクタです。
