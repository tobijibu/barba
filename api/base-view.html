Barba.BaseView

BaseViewを拡張します。


メンバ変数

Barba.BaseView.namespace

Viewの名前空間です。 (コンテナのdata-namespaceに関連付ける必要があります。)

type : string

関数

Barba.BaseView.extend(newObject)

オブジェクトを拡張するヘルパーです。 戻り値はobjectです。

newObject : object
-

Barba.BaseView.init()

ビューを初期化します。 Barba.Pjax.start()の前にビューを初期化することをお勧めします。 そうすることで、ページがロードされた時に、 コンテナに対して.onEnter()と.onEnterCompleted()が実行されます。


Barba.BaseView.onEnter()

コンテナが準備されDOMに追加された時に呼び出されます。


Barba.BaseView.onEnterCompleted()

コンテナの遷移が終了した時に呼び出されます。


Barba.BaseView.onLeave()

新しいコンテナへの遷移が開始した時に呼び出されます。


Barba.BaseView.onLeaveCompleted()

コンテナがDOMから削除された時に呼び出されます。
